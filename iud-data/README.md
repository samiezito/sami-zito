# IUD Data Analysis

For my Verbal Data Analysis final project, I set out to investigate online public discourse surrounding the procedure of IUD insertions. Using a convenience sample to collect patient data from r/birthcontrol on Reddit and provider data from health-based websites, the data was analyzed based on the type of experience and language used. A chi-square analysis was conducted. Differences in patient and provider language were found. 

Overall, patients tend to describe more negative experiences and use objective language to describe these experiences. Providers describe more neutral experiences and use hedging language to describe neutral and negative experiences. These differences highlight the need for transparent and accurate medical information so patients can make better informed decisions.
