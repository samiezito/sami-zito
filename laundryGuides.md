# NC State Laundry Guide: Task-Based Documentation for First-Time Laundry Users

## Overview
This project provides structured, user-centered documentation to help college students successfully complete laundry tasks using a variety of residential setups, including communal laundry rooms, in-unit machines, and off-campus facilities.

The guide was designed to support first-time users by breaking down the laundry process into clear, task-based instructions and adapting content to different user scenarios.

## Audience
- College students with little or no prior laundry experience  
- Residents using shared or unfamiliar laundry facilities  
- Users navigating different machine types and payment systems  

## Problem
Many students enter college without experience using laundry machines, particularly in communal or shared environments. Existing instructions are often:
- not geared toward users of communal machines 
- lacking step-by-step guidance  
- not tailored to the specific machine used in the residence hall  s

This creates confusion and errors when completing basic tasks.

## Documentation Strategy

### Scenario-Based Content Design
Created three versions of the guide to reflect different user environments:
- Communal laundry facilities  
- Apartment (in-unit) machines  
- Off-campus laundry settings  

This approach ensures users receive relevant instructions based on their specific context.

### Task-Based Organization
- Broke the laundry process into clear, actionable steps  
- Structured content to support task completion from start to finish  
- Reduced cognitive load for first-time users  

### Plain Language
- Simplified technical and procedural language  
- Prioritized clarity and accessibility
